## Overview
A Spotify-inspired music app built to explore modern iOS architecture 
and concurrency patterns, integrating both the Spotify API and iTunes API.

## Tech Highlights
- **SwiftUI** 
- **Swift Concurrency** (async/await)
- **MVVM** — clean separation of view and business logic
- **Swift Testing** — unit tests with the new Swift Testing framework
- **Spotify API + iTunes API** — dual API integration with OAuth flow
